# Home Active Directory Lab 🏗️

Documentation of my local lab environment used to simulate real-world enterprise attacks.

### 🖥️ Architecture
- **Domain Controller:** Windows Server 2019.
- **Endpoints:** Windows 10 & Windows 11 (vulnerable configurations).
- **Attacker:** Kali Linux.
- **Hypervisor:** VMware Workstation / Proxmox.

### 🧪 Scenarios Tested
- GPO abuse and misconfigurations.
- LLMNR/NBT-NS Poisoning.
- Multi-forest trust relationships and domain dominance.
